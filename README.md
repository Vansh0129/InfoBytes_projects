# 🧩  Library Management System

### Overview

This project demonstrates a **backend REST API system with authentication and API documentation**.

It includes database modeling and API testing using Swagger.

### Technologies Used

* Java
* Spring Boot
* Spring Security
* JWT Authentication
* MySQL
* DB Design
* Swagger UI

### Features

* User authentication
* Secure API endpoints
* Refresh token mechanism
* Stateless and Faster
* RESTful API architecture
* API documentation using Swagger(For Fast Lookup to all apis)
